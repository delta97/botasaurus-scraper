This walkthrough installs the studio, runs an AI task against a real form, and
saves the result as a recipe you can replay forever.

## Prerequisites

<AccordionGroup>
  <Accordion title="Python 3.9+ and Node 18+" icon="code">
    The backend is FastAPI (Python); the frontend builds with Vite (Node). Check
    with `python3 --version` and `node --version`.
  </Accordion>
  <Accordion title="Google Chrome or Chromium" icon="chrome">
    Botasaurus drives a real Chrome and does **not** download it on Linux. The
    setup script installs `google-chrome-stable` for you on Debian/Ubuntu; on
    other platforms make sure `google-chrome` or `chromium` is on your `PATH`.
  </Accordion>
  <Accordion title="An OpenRouter API key" icon="key">
    Only needed for **AI tasks** and **self-healing** — recipe replay, testing,
    datasets, and scheduling all run without one. Grab a key at
    [openrouter.ai](https://openrouter.ai).
  </Accordion>
</AccordionGroup>

## 1. Install

<Steps>
  <Step title="Clone and run setup">
    ```bash
    git clone https://github.com/delta97/botasaurus-scraper
    cd botasaurus-scraper
    ./setup.sh
    ```
    `setup.sh` vendors Botasaurus, installs the Python and frontend
    dependencies, installs Chrome if it's missing, and builds the frontend.
  </Step>
  <Step title="Start the server">
    ```bash
    uvicorn backend.main:app --host 0.0.0.0 --port 8000
    ```
    Open [http://localhost:8000](http://localhost:8000). Visit
    `http://localhost:8000/api/health` to confirm Chrome was found:
    ```json
    { "ok": true, "chrome_found": true, "chrome_path": "/usr/bin/google-chrome" }
    ```
  </Step>
  <Step title="Add your OpenRouter key">
    Open **Settings**, paste your API key, and pick a model (the dropdown is
    populated live from OpenRouter — filter for `claude`, `gpt`, or `free`).
  </Step>
</Steps>

<Warning>
In a container or on a server, keep **headless** enabled (it's the default). The
studio adds `--no-sandbox`/`--disable-dev-shm-usage` automatically when running
as root.
</Warning>

## 2. Run your first AI task

Go to **New Task** and enter a goal and a starting URL. We'll use a stable public
form so there's nothing to flake:

<CodeGroup>
```text Goal
Fill out the pizza order form with name Ada Lovelace, telephone 555-0100,
choose a medium pizza with bacon, and submit it.
```
```text Starting URL
https://httpbin.org/forms/post
```
</CodeGroup>

Hit **Run task** and watch the live timeline. You'll see the agent:

1. **Navigate** to the page (no AI — the first step is deterministic).
2. **Snapshot** the page into a compact list of interactive elements.
3. Decide a **`fill_form`** batch and type every field in one shot.
4. **Click submit** and **finish** with a summary.

Every step shows its selector, a screenshot, and timing. Expand any **LLM call**
to see the exact prompt, response, and token counts.

<Tip>
If the agent does something you didn't expect, the run timeline is your source of
truth — it's all logged to SQLite, including the model's reasoning for each action.
</Tip>

## 3. Save it as a recipe

When the run succeeds, click **Save as recipe**. The studio converts the actions
into a deterministic recipe and turns the values you supplied into
`{{variables}}`:

```yaml
name: pizza-order
variables:
  - { name: custname, default: "Ada Lovelace" }
  - { name: custtel, default: "555-0100" }
steps:
  - { type: navigate, url: "https://httpbin.org/forms/post" }
  - { type: type, selector: "input[name='custname']", value: "{{custname}}" }
  - { type: type, selector: "input[name='custtel']", value: "{{custtel}}" }
  - { type: click, selector: "input[value='medium']" }
  - { type: click, selector: "input[name='topping'][value='bacon']" }
  - { type: click, selector: "button" }
```

## 4. Replay it — no AI, no tokens

Open the recipe and hit **Replay now** with different values, or run it from the
command line:

```bash
python -m backend.runner --recipe-id 1 --var custname="Grace Hopper"
```

That replay drives Botasaurus's stealth Chrome with **zero** LLM calls.
